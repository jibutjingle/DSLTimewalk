# Installing the SDK

Start by double clicking on the `DreamscapeLearnBundle-X.X.X.exe` installer file.

<img width="484" height="328" alt="image" src="https://github.com/user-attachments/assets/cd0b19a4-d017-4786-81fc-ce9696affe2b" />

Administator privileges will be requested. Select Yes if prompted.

<img width="454" height="339" alt="image" src="https://github.com/user-attachments/assets/97567aba-5c59-4adc-99a3-1db6428fa15b" />

If prompted, install .NET SDK 5.0.408

<img width="644" height="477" alt="image" src="https://github.com/user-attachments/assets/403d04a9-a246-47fe-a6ff-5404874d1c23" />

Once successfully installed, click on the `Close` button

<img width="644" height="478" alt="image" src="https://github.com/user-attachments/assets/341e78a7-5dca-4c80-82d0-f94b26c3df60" />

If prompted, install Unity Hub

<img width="496" height="385" alt="image" src="https://github.com/user-attachments/assets/d6ace2b0-f371-4772-8d71-cc303db8ab06" />

<img width="497" height="386" alt="image" src="https://github.com/user-attachments/assets/d8f15d79-e4c0-4ca1-887a-49733662989a" />

<img width="497" height="385" alt="image" src="https://github.com/user-attachments/assets/b0eee2c5-4d29-4721-a52e-31e7ff35a12a" />

You can unselect the `Run Unity Hub` and click on `Finish`

<img width="496" height="386" alt="image" src="https://github.com/user-attachments/assets/3b6a3adc-c4b3-4a0f-aad2-0496f3f4d170" />

The Unity editor along with it’s required dependencies will now be installed. This step can take a while (up to 10 min). Do not close anything until it’s done.

Finally, the Dreamscape Learn SDK will be installed.

<img width="493" height="388" alt="image" src="https://github.com/user-attachments/assets/d379fd94-e467-4e97-a08a-55e82768d84f" />

Accept the DSL license

<img width="492" height="389" alt="image" src="https://github.com/user-attachments/assets/54bb54f4-5b78-4707-bdb2-194d8c1eb2c6" />

Install everything at default location

<img width="493" height="388" alt="image" src="https://github.com/user-attachments/assets/4705b1a4-9cc6-410f-9656-17d1081c214c" />

Click Install

<img width="491" height="389" alt="image" src="https://github.com/user-attachments/assets/30a7ea1c-bb67-49e6-8fc1-1d46ccaf3e09" />
